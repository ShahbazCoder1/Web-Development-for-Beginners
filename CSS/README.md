## CSS Attributes

1. background-color - to set a background color. You can set a solid color as well as the gradient color.

2. color - to set a text color.

3. font-size - to set font size of the text

4. font-weight - to set boldness of the text

5. margin - It is used the create a gap outside the element.

6. padding - it is used to create a gap inside the element.

7. border - this is used to set the boarder. syntax ```border: 2px solid white```

8. border-radius - to set a radius

9. height & width - to set the height and width of the element.

10. class & id - they both are used for enhance the modularity and reusability of CSS code. class is mainly used in css and id is used in js .

11. flex box: Flexbox is a layout model for arranging items (horizontally or vertically) within a container, in a flexible and responsive way.

syntax: ```display: flex;```

- flex-direction: by default is row, we can set column to align vertically.
- flex - 1: this is used to set the div to aquire how much portion on spaces.
- justify-content: Aligns the flex items when they do not use all available space on the main-axis (horizontally)
- align-items: Aligns the flex items when they do not use all available space on the cross-axis (vertically)

12. grid: The Grid Layout Module offers a grid-based layout system, with rows and columns. The Grid Layout Module allows developers to easily create complex web layouts. Syntax: ```display: grid;```
